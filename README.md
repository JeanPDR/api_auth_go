## 📖 Sobre o Projeto

Este projeto tem como objetivo fornecer um serviço robusto e seguro de autenticação. Atualmente, o projeto está em fase de construção e modelagem inicial.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Go (Golang)](https://go.dev/)
* **Banco de Dados:** [Postgres SQL](https://www.postgresql.org/)
